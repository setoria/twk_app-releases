## Downloads

Die neueste Version finden Sie im [Releases](../../releases) Bereich.

### Automatische Updates

TWK unterstützt automatische Updates. Die Software prüft beim Start automatisch auf verfügbare Updates und informiert Sie, wenn eine neue Version verfügbar ist.

Sie können auch manuell auf Updates prüfen über:
- **Hilfe** → **Auf Updates prüfen**

## Installation

1. Laden Sie die neueste Version aus dem [Releases](../../releases) Bereich herunter
2. Führen Sie den Installer aus
3. Folgen Sie den Installationsanweisungen
