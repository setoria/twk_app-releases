## Voraussetzungen

- **AxisVM** muss installiert und lizenziert sein.
- **TWK-Lizenz** (Kunden-Key) – wird beim ersten Start der App eingegeben.
- **Internetverbindung** für die Lizenzprüfung (bei jedem Start erforderlich).

---

## Sprache

Die TWK-App unterstützt mehrere Sprachen (**Deutsch**, **English**, **Italiano**, **Français**).  
Die Sprache kann jederzeit über das Menü **Sprache** gewechselt werden.

---

## Updates

Die TWK-App kann verfügbare Programm-Updates prüfen.

- Bei einer neuen Version erscheint ein Hinweisdialog mit Versionsnummer und Änderungen.
- Das Update kann direkt gestartet oder auf später verschoben werden.
- Die Installation wird geführt; danach wird die TWK-App neu gestartet.
- Für die Update-Prüfung ist eine Internetverbindung erforderlich.
