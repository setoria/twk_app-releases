# Grundriss

Im AxisVM Modellgenerator wird im Bereich **Grundriss** die Grundform des Schutzraums festgelegt.

- Grundrisstyp auswählen (z. B. rechteckig, L-förmig, U-förmig)
- Geometrieparameter eingeben
- Vorschau prüfen
